# TikTok Video Link Scraper

This Apify Actor extracts all video URLs from a given TikTok username and saves them into a `.txt` file.

## How to Use

1. Deploy the actor on Apify.
2. Enter any TikTok username in the input field.
3. Run the actor.
4. Download `video_links.txt` from the Key-Value Store.

**Note:** This actor only extracts links, no videos are downloaded. Make sure you have permission before scraping other accounts.
